# Changelog

All notable changes to ADD Framework Skills will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-01

### Added
- Initial release of universal ADD framework skills
- `add-core` - Unified overview of ADD framework
- `add-assess` - Deep Assess realm support
- `add-decide` - Deep Decide realm support (includes Livelines vs Deadlines)
- `add-do` - Deep Do realm support (includes Livelines concept)
- `add-imbalance` - Five imbalance patterns with support strategies
- `add-realm-detection` - Centralized detection patterns

### Notes
- Skills are environment-agnostic (no Claude Code or Cowork-specific features)
- Designed for embedding via git subtree
- Based on 15 years of ADD framework development by Dragos Roua
